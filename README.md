# 🍽️ Dishly

Your global kitchen companion — 21 recipes, AI Fridge Scan, calorie tracker, meal planner.

---

## 🚀 Deploy to Vercel (Free — takes ~10 minutes)

### Step 1 — Create a GitHub account
Go to **github.com** and sign up for free if you don't have one.

### Step 2 — Create a new repository
1. Click the **+** button in the top right → **New repository**
2. Name it `dishly`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload these files
1. Click **uploading an existing file** on the repository page
2. Drag and drop the entire `dishly` folder contents:
   - `package.json`
   - `vercel.json`
   - `public/index.html`
   - `src/index.js`
   - `src/App.jsx`
3. Click **Commit changes**

### Step 4 — Deploy on Vercel
1. Go to **vercel.com** and sign up with your GitHub account
2. Click **Add New Project**
3. Select your `dishly` repository
4. Click **Deploy**
5. Wait ~2 minutes for the build to finish

### Step 5 — You're live! 🎉
Vercel gives you a free URL like:
**https://dishly.vercel.app**

Share this link with anyone — it works on phones and computers!

## 💡 Custom Domain (Optional)
If you want **dishly.com** instead of the Vercel URL:
1. Buy the domain on **Namecheap** or **GoDaddy** (~$10/year)
2. In Vercel, go to Settings → Domains → Add your domain
3. Follow the DNS instructions

---

## 🔧 Making Changes
After deployment, if you want to update the app:
1. Edit `src/App.jsx` on GitHub directly
2. Vercel automatically rebuilds and deploys within 2 minutes
